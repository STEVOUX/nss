Repository Purpose
This repository is a public redirect gateway for the official NSS
Memories website.
Its only responsibility is to redirect visitors to the live experience
while keeping the complete application in a private repository.
The website itself is a digital time capsule created to preserve the
complete journey of ST. JOSEPH'S H.S.S. MATHILAKAM -- NSS Unit No. 697
(2025--2027).
---
The Vision
> Some memories are too valuable to disappear with time.
Our journey is more than photographs.
It is every camp, every act of service, every friendship, every laugh,
every challenge and every achievement woven together into one story.
The official website transforms those moments into a timeless digital
archive for present and future generations.
